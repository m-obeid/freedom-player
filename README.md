# Freedom Player
A simple application to play YouTube videos without YouTube Ads, no Adblock required! Is much more resource friendly than a Adblock too!

# Source Code
You can freely use the source code of this application. Make sure to agree tu the GNU GPLv3 License! (LICENSE.TXT)

# Download
Under Releases you should be able to download the Setup EXE files or the Manual Installation ZIP file.
Or go on https://pocoguy.tk/freedom

# Programming Language
I used C#.NET Framework 4.7.2 Windows Form Applications for the application.
Libaries that were used also are:

1. ActiveX VLC Plugin 2 (for playing YouTube videos thats the only I found)
2. CefSharp.Winforms (for the freedom browser)
